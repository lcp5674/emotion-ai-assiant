## 项目概述

AI情感助手是一款基于人工智能的情感支持与个人成长平台，为用户提供专业的心理测评、人格分析、情感咨询等服务。

### 核心功能

| 模块 | 功能 | 状态 |
|------|------|------|
| MBTI测评 | 16型人格测试与解读 | ✅ 已完成 |
| SBTI测评 | 情感风格测评（34主题） | ✅ 已完成 |
| 依恋风格 | 成人依恋类型评估 | ✅ 已完成 |
| 深度画像 | 三位一体人格综合分析 | ✅ 已完成 |
| AI对话 | RAG增强的智能对话 | ✅ 已完成 |
| 数据质量 | 时效性/完整性/一致性保障 | ✅ 已完成 |

## 产品需求文档

> 完整内容请查看 `AI情感助手PRD.md`

### 用户群体

- 18-35岁年轻人
- 对自我认知有兴趣的用户
- 需要情感支持的用户
- 追求个人成长的用户

### 核心场景

1. **人格探索**：通过MBTI了解自己的性格特点
2. **情感风格**：通过SBTI了解情感表达方式
3. **依恋模式**：了解自己在关系中的行为模式
4. **深度洞察**：整合多维度数据获得个性化分析
5. **AI陪伴**：获得7x24小时的智能情感支持

### 用户旅程

```
注册 → 完成MBTI → 完成SBTI → 完成依恋测试 → 生成深度画像 → AI对话
```
