## 四、架构优势

### 4.1 工厂模式统一管理
```python
# LLM使用示例
from app.services.llm.factory import get_llm_provider, list_available_providers
provider = get_llm_provider()

# Embedding使用示例
from app.services.embedding_service import EmbeddingFactory
embedding = EmbeddingFactory.get_embedding()
```

### 4.2 配置驱动
所有AI厂商配置通过`config.py`管理，支持环境变量：
```bash
# LLM配置示例
LLM_PROVIDER=volcengine
VOLCENGINE_API_KEY=your_key
VOLCENGINE_MODEL=doubao-pro-32k

# Embedding配置示例
LLM_PROVIDER=sensetime  # 将使用商汤Embedding
SENSETIME_API_KEY=your_key
```

### 4.3 生产级特性
- **重试机制**: 指数退避，最多重试3次
- **结果验证**: 自动验证返回结果完整性
- **错误日志**: 详细的错误日志记录
- **零向量回退**: API失败时返回零向量，保证系统可用性

## 六、火山引擎支持详情

### 6.1 为什么需要火山引擎支持

1. **字节跳动官方云平台**: 豆包模型官方部署平台
2. **国内合规**: 数据存储在国内，满足合规要求
3. **性能优化**: 针对国内网络优化，延迟更低
4. **成本优势**: 国内定价，本地化计费

### 6.2 配置方式

```bash
# 火山引擎LLM
LLM_PROVIDER=volcengine
VOLCENGINE_API_KEY=your_ark_api_key
VOLCENGINE_MODEL=doubao-pro-32k

# 火山引擎Embedding
LLM_PROVIDER=volcengine  # Embedding使用同一provider
```

### 6.3 API端点
- LLM: `https://ark.cn-beijing.volces.com/api/v3/chat/completions`
- Embedding: `https://ark.cn-beijing.volces.com/api/v3/embeddings`
