## 📝 获取火山引擎 API Key

### 步骤 1: 注册火山引擎账号

1. 访问 [火山引擎官网](https://www.volcengine.com/)
2. 点击"立即注册"
3. 选择"个人用户"或"企业用户"
4. 完成手机号验证

### 步骤 2: 实名认证

1. 登录火山引擎控制台
2. 进入"账号中心" → "实名认证"
3. 完成个人实名认证（支持支付宝快速认证）

### 步骤 3: 开通大模型服务

1. 访问 [火山引擎 ARK 控制台](https://console.volcengine.com/ark)
2. 点击"开通服务"
3. 阅读并同意服务协议
4. 完成开通

### 步骤 4: 创建 API Key

1. 进入"API Key 管理"
2. 点击"创建 API Key"
3. 输入名称（如：emotion-ai）
4. 点击"创建"
5. **立即复制保存 API Key**（只显示一次！）

### 步骤 5: 获取免费额度

火山引擎提供免费试用额度：
- 注册即送 **10 元**免费额度
- 相当于 **100 万 tokens**
- 足够开发测试使用

在 `backend/.env` 中添加或修改：

```bash

```

## 🧪 完整配置示例

创建 `backend/.env` 文件：

```bash
# 应用配置
APP_NAME=心灵伴侣AI
APP_VERSION=1.0.0
DEBUG=true

# 服务器
HOST=0.0.0.0
PORT=8000

# 数据库（轻量测试使用 SQLite）
DATABASE_URL=sqlite:///./emotion_ai.db

# Redis（开发环境可选）
REDIS_HOST=localhost
REDIS_PORT=6379
REDIS_PASSWORD=
REDIS_DB=0

# JWT
SECRET_KEY=dev-secret-key-change-in-production-1234567890
ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=10080

# ============================
# 火山引擎豆包大模型
# ============================
LLM_PROVIDER=doubao
DOUBAO_API_KEY=your-api-key-here
DOUBAO_MODEL=doubao-pro-32k
DOUBAO_BASE_URL=https://ark.cn-beijing.volces.com/api/v3

# CORS
CORS_ORIGINS=["http://localhost:5173","http://localhost:3000"]
```

## 🔧 常见问题

### Q1: API Key 在哪里找？

1. 登录 [火山引擎控制台](https://console.volcengine.com/)
2. 进入"ARK 大模型服务平台"
3. 点击左侧"API Key 管理"
4. 创建或复制现有 API Key

### Q2: Region 选择哪个？

推荐使用**北京 Region**：

```bash
DOUBAO_BASE_URL=https://ark.cn-beijing.volces.com/api/v3
```

其他可用 Region：
- `ark.cn-beijing.volces.com` - 北京（推荐）
- `ark.ap-guangzhou.volces.com` - 广州

### Q3: 模型名称怎么选？

开发测试推荐：
```bash
DOUBAO_MODEL=doubao-lite-32k  # 便宜、快速
```

生产环境推荐：
```bash
DOUBAO_MODEL=doubao-pro-32k  # 性能强、稳定
```

### Q4: 调用失败怎么办？

1. 检查 API Key 是否正确
2. 确认是否开通了对应模型
3. 查看余额是否充足
4. 检查网络连接

```bash
# 测试 API Key 是否有效
curl https://ark.cn-beijing.volces.com/api/v3/models \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### Q5: 如何监控使用量？

1. 登录火山引擎控制台
2. 进入"费用中心" → "用量明细"
3. 查看 API 调用统计
