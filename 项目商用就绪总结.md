## 技术架构

### 后端技术栈
- **框架**：FastAPI 0.104.1
- **数据库**：SQLAlchemy + MySQL
- **认证**：OAuth2 + JWT
- **AI集成**：支持多种大模型（OpenAI/Claude/国产模型）
- **向量搜索**：Milvus + Sentence-Transformers

### 前端技术栈
- **框架**：React 18.2.0
- **UI库**：Ant Design 5.11.5
- **状态管理**：Zustand
- **构建工具**：Vite 5.0.4
- **路由**：React Router DOM 6.20.1

### 部署架构
- **容器化**：Docker + Docker Compose
- **服务编排**：Docker Compose 3.8
- **服务列表**：
  - 后端API服务
  - 前端Web应用
  - MySQL数据库
  - Redis缓存
  - Milvus向量数据库

## 部署指南

### 本地开发环境

**启动后端**：
```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

**启动前端**：
```bash
cd frontend/web
npm install
npm run dev
```

### Docker部署

```bash
# 启动所有服务
docker-compose up -d

# 查看日志
docker-compose logs -f

# 停止服务
docker-compose down
```

### 生产环境部署

**部署前检查清单**：
- [ ] 配置强 SECRET_KEY（至少32字符）
