## 一、项目概述

### 1.1 项目定位

**心灵伴侣AI** - 懂你的AI情感伴侣

- **核心功能**: MBTI人格测试 + AI情感对话 + RAG知识库
- **目标用户**: 职场新人、大学生、情感困惑者、自我成长者
- **商业模式**: 免费+会员制(月卡/年卡/终身会员)

### 1.2 技术栈

| 层级 | 技术选型 | 版本 |
|------|----------|------|
| **前端** | React + TypeScript + Ant Design + Zustand | React 18, TypeScript 5 |
| **后端** | Python + FastAPI + SQLAlchemy | Python 3.10+, FastAPI 0.109 |
| **数据库** | MySQL 8.0 + Redis 7 | MySQL 8.0 |
| **AI层** | RAG + 多LLM支持 | OpenAI/Claude/GLM/Qwen/MiniMax等 |
| **部署** | Docker + Docker Compose | - |

### 1.3 项目结构

class ValidationException(AppException):
    code = "VALIDATION_ERROR"
    status_code = 400

    status_code = 404
