# 管理后台使用指南

## 🚀 管理后台功能概览

项目已内置**完整的管理后台**，支持界面化配置所有服务！

---

## 📊 功能模块

### 1️⃣ LLM 大模型配置（核心功能）

**支持 11 种大模型**：

| 模型 | 标识符 | 配置项 |
|------|--------|--------|
| Mock | `mock` | 开发测试 |
| OpenAI GPT | `openai` | API Key + 模型 |
| Anthropic Claude | `anthropic` | API Key + 模型 |
| 智谱 GLM | `glm` | API Key + 模型 |
| 通义千问 | `qwen` | API Key + 模型 |
| MiniMax | `minimax` | API Key + 模型 |
| 百度文心一言 | `ernie` | API Key + 模型 |
| 腾讯混元 | `hunyuan` | API Key + 模型 |
| 讯飞星火 | `spark` | API Key + 模型 |
| **火山引擎豆包** | `doubao` | API Key + 模型 |
| 硅基流动 | `siliconflow` | API Key + 模型 |

**配置界面**：
```typescript
// 前端配置表单
interface LLMConfig {
  llm_provider: 'doubao'  // 选择模型
  doubao_api_key: string   // API密钥
  doubao_model: string    // 模型名称
  doubao_base_url?: string // API地址（可选）
}
```

### 2️⃣ AI 助手管理

**功能**：
- ✅ 创建助手（名称、MBTI、性格、问候语）

**助手配置**：
```typescript
interface Assistant {
  name: string           // 助手名称
  mbti_type: string     // MBTI 类型
  personality?: string  // 性格描述

  greeting?: string     // 问候语
  expertise?: string     // 专业领域
  is_active: boolean    // 是否启用
  is_recommended: boolean // 是否推荐
}
```

## 🔧 核心 API 接口

### LLM 配置管理

```bash
# 获取当前配置
GET /api/v1/admin/config
Response: {
  "llm_provider": "doubao",
  "doubao_api_key": "***",
  "doubao_model": "doubao-pro-32k",
  "has_doubao_key": true
}

# 更新配置
POST /api/v1/admin/config
Body: {
  "llm_provider": "doubao",
  "doubao_api_key": "your-key",
  "doubao_model": "doubao-pro-32k"
}

# 测试连接
POST /api/v1/admin/test
Body: {
  "provider": "doubao",
  "api_key": "your-key",
  "model": "doubao-pro-32k"
}
Response: {
  "success": true,
  "message": "连接成功"
}
```

### AI 助手管理

```bash
# 获取助手列表
GET /api/v1/admin/assistants

# 创建助手
POST /api/v1/admin/assistants
Body: {
  "name": "心灵导师",
  "mbti_type": "ENFP",
  "personality": "温暖、乐观、有同理心",
  "speaking_style": "亲切、自然",

Body: {

  "is_active": true

# 删除助手
DELETE /api/v1/admin/assistants/{id}
```

## ⚙️ 配置流程（界面化）

### 步骤 1: 登录管理后台

```
1. 访问 http://localhost:5173/admin
2. 使用管理员账号登录
   - 如果没有管理员账号，需要在数据库中创建
   - 或联系系统管理员
```

### 步骤 2: 配置大模型

```
1. 点击左侧菜单 "系统配置"
2. 在 "大模型配置" 部分：
   - 选择 Provider（火山引擎豆包）
   - 输入 API Key
   - 选择或输入 Model
3. 点击 "测试连接" 验证配置
4. 点击 "保存配置"
```

### 步骤 3: 管理 AI 助手

```
1. 点击左侧菜单 "助手管理"
2. 可以：
   - 查看现有助手
   - 创建新助手
   - 编辑助手配置
   - 删除不需要的助手
```

## 💡 最佳实践

### 1. 安全建议

- ✅ **API Key 安全**：
  - 管理后台输入的 Key 会加密存储
  - 建议定期更换 API Key
  - 不要在公开场合泄露

- ✅ **权限控制**：
  - 只有管理员可以访问 `/admin` 路由
  - 建议为不同管理员分配不同权限

### 2. 配置建议

- ✅ **开发环境**：使用 Mock 模型
- ✅ **测试环境**：使用智谱 GLM（便宜）
- ✅ **生产环境**：使用火山引擎豆包（稳定）

### 3. 监控建议

- ✅ **定期查看统计**：
  - 监控用户增长
  - 监控 API 使用量
  - 监控付费转化

- ✅ **日志查看**：
  ```bash
  tail -f logs/backend.log
  ```

## 🐛 故障排查

### 问题 1: 无法访问 /admin

**原因**：需要管理员权限

**解决**：
```bash
# 检查用户是否是管理员
sqlite3 emotion_ai.db
sqlite> SELECT is_admin FROM users WHERE phone = '13800138000';

# 设置管理员
sqlite> UPDATE users SET is_admin = 1 WHERE phone = '13800138000';
```

### 问题 2: LLM 配置保存失败

**原因**：配置验证失败

**解决**：
1. 检查 API Key 是否正确
2. 点击 "测试连接" 查看错误信息
3. 查看后端日志：`tail -f logs/backend.log`

### 问题 3: 模型不响应

**原因**：API Key 无效或额度用完

**解决**：
1. 登录对应平台检查 API Key
2. 检查账户余额/免费额度
3. 查看错误日志
