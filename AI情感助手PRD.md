## 1. 产品概述

### 1.1 产品背景

在快节奏的现代生活中，人们面临着越来越多的情感困惑、人际关系挑战和心理健康问题。传统的心理咨询服务存在成本高、预约困难、隐私顾虑等痛点，而通用的AI聊天机器人又缺乏针对性和专业性。

### 1.2 产品简介

**心灵伴侣**是一款基于RAG（检索增强生成）+ AI大模型的个性化情感助手产品。用户通过完成MBTI人格测试，系统根据用户的人格类型、情感状态和咨询需求，匹配专属的AI情感助手，提供有针对性的情感咨询、心理疏导和人际关系指导服务。

**核心差异化**：
- 基于MBTI人格类型的个性化服务
- 集成多种心理学模型的专业支撑
- RAG知识库确保回答的专业性和准确性
- 情感陪伴与专业疏导结合

### 7.2 检索流程

```
用户问题 → 向量化 → 相似度检索 → 知识库 → Top-K结果 → 上下文组装 → LLM生成
```

### 7.3 检索优化策略

1. **向量语义检索**：语义相似度匹配
2. **关键词检索**：精确匹配补充
3. **MBTI类型过滤**：按人格类型筛选
4. **混合排序**：多路结果融合

### 9.3 RAG技术架构

```python
# 核心检索逻辑
class RAGService:
    def retrieve(self, query, user_mbti, top_k=5):
        # 1. 向量检索
        semantic_results = self.vector_store.search(query, top_k=10)

        # 2. 关键词检索
        keyword_results = self.es.search(query, top_k=10)

        # 3. MBTI过滤
        filtered = self.filter_by_mbti(semantic_results, user_mbti)

        # 4. 混合排序
        results = self.rerank([filtered, keyword_results])

        return results[:top_k]

    def generate(self, query, context, user_profile):
        # 构建提示词
        prompt = self.build_prompt(query, context, user_profile)

        # 调用大模型
        response = self.llm.chat(prompt)

        return response
```

## 10. 业务流程

### 10.1 用户旅程

```
发现 → 注册 → MBTI测试 → 匹配助手 → 对话互动 → 付费转化 → 持续使用 → 推荐分享
```

### 10.2 核心流程

#### 10.2.1 MBTI测试流程

```
1. 用户进入测试页面
2. 阅读测试说明
3. 回答48道选择题
4. 系统计算四个维度倾向
5. 生成16型人格结果
6. 展示详细分析报告
7. 推荐匹配AI助手
8. 用户激活助手开始对话
```

#### 10.2.2 AI对话流程

```
1. 用户输入问题/消息
2. 系统识别意图和情绪
3. 检索RAG知识库
4. 结合用户MBTI类型
5. 大模型生成回复
6. 返回个性化回答
7. 更新对话上下文
```

#### 15.3.3 模型路由策略

| 策略 | 说明 | 适用场景 |
|------|------|----------|
| 固定模型 | 统一使用一个模型 | 简单场景 |

**智能路由示例**：

```python
def route_model(question_type: str, user_tier: str) -> str:
    """智能路由"""
    if user_tier == 'free':
        # 免费用户使用性价比高的模型
        return 'qwen-turbo'

    if question_type == 'emotional_support':
        # 情感支持使用情商更高的模型
        return 'glm-4'

    if question_type == 'professional':
        # 专业问题使用能力更强的模型
        return 'qwen-max'

    # 默认使用均衡模型
    return 'glm-4-turbo'
```
