## 运行测试

```bash
cd backend

# 运行所有测试
pytest tests/ -v

# 运行单元测试
pytest tests/unit/ -v --cov=app

# 运行集成测试
pytest tests/integration/ -v

# 生成覆盖率报告
pytest tests/ --cov=app --cov-report=html
```
