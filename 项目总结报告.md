## 技术架构

### 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | React 18 + TypeScript | 单页应用 |
| 状态管理 | Zustand | 轻量级状态管理 |
| UI框架 | Ant Design | 企业级组件库 |
| 后端 | FastAPI | 高性能Python框架 |
| 数据库 | MySQL 8.0 | 关系型数据 |
| 缓存 | Redis 7 | 缓存和会话 |
| 对象存储 | MinIO | 文件存储 |
| AI服务 | RAG + LLM | 知识增强对话 |

### 项目结构

```
emotion-ai-assiant/
├── backend/                 # 后端服务
│   ├── app/
│   │   ├── api/v1/         # API路由
│   │   ├── models/         # 数据模型
│   │   ├── schemas/        # Pydantic模型
│   │   ├── services/       # 业务服务
│   │   └── services/rag/   # RAG服务
│   ├── tests/              # 测试
│   └── requirements.txt
├── frontend/web/            # 前端应用
│   ├── src/
│   │   ├── api/            # API调用
│   │   ├── stores/         # 状态管理
│   │   ├── views/          # 页面组件
```
