# SBTI模块质量检查清单

## 概述
SBTI（优势才干测评）模块是AI情感助手的人格画像核心组成部分，与MBTI、依恋风格共同构成"三位一体深度人格画像"。

## 功能测试

### 1. SBTI优势才干测评
- [ ] **24道题目正确加载**
  - 题目数量：24题
  - 题目状态：全部为启用状态
  - 题目格式：每题为二选一选择
  - 选项对应才干：A/B选项对应不同才干主题
  
- [ ] **二选一选择正确记录**
  - 用户选择被正确记录
  - 答案映射到正确的才干主题
  - 权重计算正确（权重A/权重B）
  
- [ ] **才干分数正确计算**
  - 34项才干分数正确计算
  - TOP5优势才干正确排序
  - 分数无越界错误
  
- [ ] **四大领域分布计算**
  - 执行力领域得分计算
  - 影响力领域得分计算  
  - 关系建立领域得分计算
  - 战略思维领域得分计算
  
- [ ] **结果正确保存**
  - SBTI结果保存到数据库
  - 与用户正确关联
  - 版本控制正确（is_latest字段）
  - 历史记录保存完整

### 2. 依恋风格测评
- [ ] **10道题目正确加载**
  - 题目数量：10题
  - 量表范围：1-7分李克特量表
  - 维度权重：焦虑维度、回避维度权重正确
  
- [ ] **维度得分计算**
  - 焦虑维度总分计算正确
  - 回避维度总分计算正确
  - 平均分计算正确
  
- [ ] **依恋风格分类**
  - 安全型：低焦虑+低回避（得分均<3.5）
  - 焦虑型：高焦虑+低回避（焦虑≥3.5，回避<3.5）
  - 回避型：低焦虑+高回避（焦虑<3.5，回避≥3.5）
  - 混乱型：高焦虑+高回避（得分均≥3.5）
  
- [ ] **个性化建议生成**
  - 关系建议与风格匹配
  - 自我成长建议合理
  - 特征标签准确
